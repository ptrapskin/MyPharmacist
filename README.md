# MyPharmacist

Static site files for the MyPharmacist project.

## Local preview

Open [index.html](./index.html) in a browser.

## Repository structure

- `index.html`
- `styles.css`
- `taper-calendar.html`
- `favicon.svg`

## GitHub Pages

This repository includes a GitHub Actions workflow that publishes the site to GitHub Pages whenever changes are pushed to `main`.

After creating the GitHub repository:

1. Push this project to the `main` branch.
2. In GitHub, open `Settings > Pages`.
3. Confirm the source is set to `GitHub Actions`.
